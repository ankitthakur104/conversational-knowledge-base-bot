✨Key Features

Multi-turn Memory

ConversationBufferWindowMemory (k=5) maintains per-session context with zero cross-session leakage.

Confidence Scoring

Lexical overlap between generated answer and retrieved context flags low-confidence answers.

Hallucination Guardrails

Low-confidence answers trigger graceful fallback and escalate to human support automatically.

RAG over 50K+ Docs

BM25 + Pinecone retrieval over company knowledge base with per-product namespaced collections.

Session Isolation

Independent memory state per session ID — safe for high-concurrency multi-user deployments.

FastAPI + Docker

Containerized REST API with streaming support and horizontal scaling ready.

def score_confidence(answer: str, docs: list) -> float:
      a_words = set(answer.lower().split())
      c_words = set(" ".join(d.page_content for d in docs).lower().split())
      return min(len(a_words & c_words) / max(len(a_words), 1), 1.0)
